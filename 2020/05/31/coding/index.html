    用Hexo+Coding搭建一个访问更快的个人blog

    工具 hexo Coding
    发布时间 : 2020-05-31 08:40
    阅读 :
    1. 起由
    2. 构建流程

    起由

    • Hexo+GitHub Pages搭建出的自己的blog，访问起来慢。

    构建流程

    • 技术：Hexo+Coding

      • 流程：如果在gitHub部署过，只需要在Coding注册个团队，自己创建个项目。

      • 记得添加ssh key，和git一样,如果git新增过公钥，只需要找到本地文件夹复制，然后打开coding -> 个人设置 -> ssh公钥 ->新增，如果git没有新增过可以看下面相关链接.

      • 相关链接

        • Hexo博客框架的搭建的相关操作：Hexo建站手册（详细教程）
        • Hexo部署到Coding上
        • Coding
        • Hexo官网地址

      • 然后我的团队 -> 全部项目 -> 进入到刚才创建的项目 -> 项目概述 -> 复制右侧链接 -> 去自己项目的_config.yml文件中 修改 repository：地址

        deploy:
        type: git
        repository: https://github.com/liusheng-zhainanya/liusheng-zhainanya.github.io.git # 用https或者ssh均可
        repository: https://e.coding.net/liusheng-zhainanya/ls-blog.git # 用https或者ssh均可
        branch: master
